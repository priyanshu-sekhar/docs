Parameta's OTC market data is available through **LSEG's** (formerly Refinitiv) enterprise data platforms, providing seamless integration with your existing LSEG infrastructure.

## Available Products

### LSEG DataScope

**DataScope** is LSEG's comprehensive data extraction and delivery platform.

| Feature | Description |
| --- | --- |
| **End-of-Day Data** | Daily snapshots and closing prices |
| **Time Series** | Historical data for backtesting |
| **Bulk Extraction** | Large dataset downloads via API or SFTP |

### LSEG RTDS (Real-Time Distribution System)

**RTDS** is LSEG's enterprise real-time data distribution platform.

| Feature | Description |
| --- | --- |
| **Real-time Streaming** | Live tick-by-tick market data |
| **Low Latency** | Optimized for trading applications |
| **Enterprise Scale** | Designed for large-scale data distribution |

## Parameta Data Available

Through LSEG, you can access Parameta's:

- **Interest Rate Derivatives** — Swaps, swaptions, caps/floors
- **FX Options** — Currency options and volatility data
- **Credit Derivatives** — CDS spreads and indices
- **Commodities** — Energy and commodity derivatives

## Integration

Parameta data through LSEG integrates with:

- **Eikon** — LSEG's desktop terminal
- **Workspace** — LSEG's next-generation platform
- **Enterprise APIs** — Direct API access for custom applications
- **Third-party Systems** — OMS, EMS, and risk systems

## Getting Started

1. **Contact LSEG** — Reach out to your LSEG representative
2. **Request Parameta Feed** — Specify the Parameta data products you need
3. **Configure Access** — Set up DataScope or RTDS delivery
4. **Start Consuming** — Access data through your preferred LSEG interface
