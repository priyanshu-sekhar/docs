Parameta's OTC market data is available through **Bloomberg's** enterprise data distribution platforms, enabling seamless integration with your existing Bloomberg infrastructure.

## Available Products

### Bloomberg B-Pipe

**B-Pipe** is Bloomberg's enterprise data feed solution for real-time market data distribution.

| Feature | Description |
| --- | --- |
| **Real-time Data** | Live tick-by-tick OTC market data |
| **Low Latency** | Sub-second data delivery |
| **Integration** | Direct feed into trading systems and applications |

### Bloomberg Data License

**Data License** provides access to Bloomberg's comprehensive data sets for analytics and research.

| Feature | Description |
| --- | --- |
| **Historical Data** | Access to time series data |
| **Reference Data** | Instrument and entity reference data |
| **Bulk Delivery** | Large dataset downloads |

## Parameta Data Available

Through Bloomberg, you can access Parameta's:

- **Interest Rate Derivatives** — Swaps, swaptions, caps/floors
- **FX Options** — Currency options and volatility data
- **Credit Derivatives** — CDS spreads and indices
- **Commodities** — Energy and commodity derivatives

## Getting Started

1. **Contact Bloomberg** — Reach out to your Bloomberg representative
2. **Request Parameta Feed** — Specify the Parameta data products you need
3. **Configure Terminal/API** — Set up access through B-Pipe or Data License
4. **Start Consuming** — Access data through Bloomberg Terminal or API
