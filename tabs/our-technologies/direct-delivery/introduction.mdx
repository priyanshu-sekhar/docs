Parameta delivers exclusive market data, indices, and analytics solutions in a way that best suits your existing workflows. Available through a growing list of partners including third-party vendors and cloud services such as **AWS** and **Snowflake**, as well as via API feeds or our proprietary technology solutions.

This document is focused on **Direct Delivery** methods. Direct delivery methods available can differ based on whether the data feed received is Real-Time (tick), Intraday (known as "snap" data), or Historical (time series).

## Delivery Options Overview

  <Card title="Direct Delivery" icon="bolt">

  <Card title="Cloud Delivery" icon="cloud">

### Real-time Streaming

Sub-second delivery of every tick of data directly into your tools:

| Method | Description |
| --- | --- |
| **WebSocket** | JSON via WebSocket for real-time data streaming |
| **Marketfeed** | API leveraging Marketfeed protocol for reliable, auditable data delivery |
| **EMD Connect** | Enterprise Market Data Distribution for direct infrastructure integration |
| **SURFIX** | FIX 5.0 SP2 compliant gateway for market data routing |

### Snap Service

Snapshots of time series data, delivered at customizable intervals and formats:

| Delivery Method | Supported Formats |
| --- | --- |
| **SFTP** | CSV, JSON, XML, XLSX, Parquet, TXT |
| **Snowflake** | Standard format |
| **Amazon S3** | All standard formats |
| **Google Cloud Storage** | All standard formats |
| **Azure Blob** | All standard formats |

### Cloud Delivery

Historical Time series and live data delivered near-real-time via Cloud Share:

| Platform | Options |
| --- | --- |
| **Snowflake Reader** | Read-only accounts for non-Snowflake customers |
| **Snowflake Direct share** | Deliverable over all major cloud providers across all regions |
| **Amazon S3** | All standard formats |
| **Google Cloud Storage** | All standard formats |
| **Azure Blob** | All standard formats |

<Note>
  For near real-time, intraday, or end-of-day use cases, direct delivery is one of our preferred methods. It's also used extensively for customer trials.
</Note>
