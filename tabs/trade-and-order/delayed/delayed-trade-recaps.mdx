## QuickStart

### Connecting to SFTP

<CodeGroup>

```python Python
import paramiko

ssh = paramiko.SSHClient()
ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
ssh.connect("HOSTNAME", port=22, username="USERNAME", password="PASSWORD")
sftp = ssh.open_sftp()

# Example: list files
print(sftp.listdir("."))

sftp.close()
ssh.close()
```

```java Java
import com.jcraft.jsch.*;

JSch jsch = new JSch();
Session session = jsch.getSession("USERNAME", "HOSTNAME", 22);
session.setPassword("PASSWORD");
session.setConfig("StrictHostKeyChecking", "no");
session.connect();

ChannelSftp sftp = (ChannelSftp) session.openChannel("sftp");
sftp.connect();

// Example: list files
sftp.ls(".").forEach(System.out::println);

sftp.disconnect();
session.disconnect();
```

</CodeGroup>

```text
<client_name>/
	└── ec_energy_gas_orders_dog_amer_view_2026-06-22.csv
```

### Delayed Files Additional Fields

| Field Name | Description | Datatype | Example |
| --- | --- | --- | --- |
| SNAP\_DATE | The date the file was snapped | Date (DD/MM/YYYY) | 4/25/2024 |
| INSERT\_TIMESTAMP | The date/time the trade was inserted into the system | Date/Time (YYYY-MM-DDThh:mm:ss.sssZ) | 2024-04-25T08:08:28.211Z |
