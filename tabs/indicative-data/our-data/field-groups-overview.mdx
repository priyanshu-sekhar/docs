# Example

## IRIRS — Interest Rates, Interest Rate Swap

| FID | DataType | Field |
| --- | --- | --- |
| `GN_TXT16_2` | `String` | Daycount |
| `GN_TXT32_4` | `String` | Instrument Description |
| `MID_PRICE` | `Real64` | Mid |
| `GN_TX20_2` | `String` | Settlement Index |
| `ACTIV_DATE` | `Date` | Date of Update |
| `BID` | `Real64` | Bid |
| `ASK` | `Real64` | Ask |
| `TIMACT` | `Time` | Time of Update |
| `PERIOD1` | `Enum` | Tenor |
| `CCY1` | `String` | Currency |
| `VALUE_DT1` | `Date` | Start Date |
| `VALUE_DT2` | `Date` | End Date |

<Warning>
  Please note that none pricing fields such as DayCount can be subject to change and we can't guarantee throughout the dataset. To avoid issues with this please use the metadata tables provided where possible e.g. `INFO.RECORDS`
</Warning>

<Warning>
  Please note that the ACTIV\_DATE and TIMACT are generated by the publisher. This means the timezone is unconfirmed and may not always be available.
</Warning>
