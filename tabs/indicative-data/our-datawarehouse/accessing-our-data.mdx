When speaking to our commercial team they will guide you on the best solution for you. We offer the two following solutions:

- [Reader Accounts](#getting-started-with-reader-accounts) - connect to Snowflake data without having a snowflake account
- [Direct Shares](#getting-started-with-direct-shares) - for existing Snowflake customers, access the data directly in your data warehouse

## Getting Started with Reader Accounts

1. **Receive Credentials** — Your Parameta sales representative will provide you with a URL and login credentials (username and password)
2. **Sign In** — Navigate to the provided URL and enter your credentials
3. **Access Data** — Once signed in, you can immediately start querying data

## Getting started with Direct Shares

At Parameta Solutions we are leveraging Snowflake's autofufillment capabilities to enable us to share across regions and cloud providers. To get started:

1. **Provide Account Your Details** — Share your [Snowflake account identifier](https://docs.snowflake.com/en/user-guide/admin-account-identifier#label-account-name-data-sharing) with your Parameta representative
2. **Accept the Share** — Once shared, accept the share in SnowSight under **Data \> Private Sharing \> Shared With You**
3. **Create Database** — Click **Get Data** to create a database from the share
4. **Start Querying** — Query the data using your preferred tools and workflows

## Connect Using SDKs

Snowflake can be queried easily using Python by using Snowflake's snowflake-connector-python. This guide provides sample code for connecting, querying data, and working with the info schema.

<Note>
  **Note:** Snowflake can be programmatically accessed via various other languages. For more information refer to [Snowflake drivers](https://docs.snowflake.com/en/developer-guide/drivers).
</Note>

### Pre-requisites

Install the Snowflake connector for Python:

<CodeGroup>

```shellscript Python
pip install "snowflake-connector-python[pandas]"
```

```shellscript Java (JBang)
# Install JBang
curl -Ls https://sh.jbang.dev | bash -s - app setup
```

</CodeGroup>

### Basic Connection

<CodeGroup>

```python Python
import snowflake.connector
import pandas as pd

connection_settings = {
    'account': '<your_account_id_and_region>',  # typically start of URL (e.g. xyz.eu-west-1)
    'user': '<your_username>',
    'password': '<your_password>',
    'database': 'commercial_packages',
    'schema': 'info'
}

with snowflake.connector.connect(**connection_settings) as ctx:
    cur = ctx.cursor()
    cur.execute("""SELECT * FROM info.records LIMIT 10""")
    df = cur.fetch_pandas_all()

df.head()
```

```java Java (JBang)
///usr/bin/env jbang "$0" "$@" ; exit $?
//DEPS net.snowflake:snowflake-jdbc:3.16.1

// Run with `jbang SnowflakeQuery.java`

import java.sql.*;
import java.util.Properties;

public class SnowflakeQuery {
    public static void main(String[] args) throws Exception {
        Properties props = new Properties();
        props.put("account", "<your_account_id_and_region>");// typically start of URL (e.g. xyz.eu-west-1)
        props.put("user", "<your_username>");
        props.put("password", "<your_password>");
        props.put("db", "commercial_packages");
        props.put("schema", "info");

        String url = "jdbc:snowflake://xyz.eu-west-1.snowflakecomputing.com";

        try (Connection conn = DriverManager.getConnection(url, props);
             Statement stmt = conn.createStatement();
             ResultSet rs = stmt.executeQuery("SELECT * FROM info.records LIMIT 10")) {

            ResultSetMetaData meta = rs.getMetaData();
            int cols = meta.getColumnCount();

            for (int i = 1; i <= cols; i++) System.out.print(meta.getColumnName(i) + "\t");
            System.out.println();

            while (rs.next()) {
                for (int i = 1; i <= cols; i++) System.out.print(rs.getString(i) + "\t");
                System.out.println();
            }
        }
    }
}
```

</CodeGroup>
