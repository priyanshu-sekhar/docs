## QuickStart

### Connecting

<CodeGroup>

```python Python
import paramiko

ssh = paramiko.SSHClient()
ssh.set_missing_host_key_policy(paramiko.AutoAddPolicy())
ssh.connect("HOSTNAME", port=22, username="USERNAME", password="PASSWORD")
sftp = ssh.open_sftp()

# Example: list files
print(sftp.listdir("."))

sftp.close()
ssh.close()
```

```java Java
import com.jcraft.jsch.*;

JSch jsch = new JSch();
Session session = jsch.getSession("USERNAME", "HOSTNAME", 22);
session.setPassword("PASSWORD");
session.setConfig("StrictHostKeyChecking", "no");
session.connect();

ChannelSftp sftp = (ChannelSftp) session.openChannel("sftp");
sftp.connect();

// Example: list files
sftp.ls(".").forEach(System.out::println);

sftp.disconnect();
session.disconnect();
```

</CodeGroup>

## SFTP Folder Structure

```text
<client_name>_pds/
	└── _information/
	    └── descriptions.xlsx
		└── indexlist.csv
	└── TPFuelOil/
		└── TPFuelOil_YYYYMMDD.csv
	└── TPFuelOil.csv
```

### File Definitions

- `_information/descriptions.xlsx` - file format definition file
- `_information/indexlist.csv` - list of indices being received as part of EoD Prices subscription
- `TPFuelOil/TPFuelOil_20251101.csv` - archived and dated file (note intraday files will also have `_HHMMSS`)
- `TPFuelOil.csv` - latest file

## File Format

#### Status Codes

| Code | Description |
| --- | --- |
| `N` | New Quotation |
| `C` | Correction |
| `B` | Backfill |
| `D` | To Delete |

#### Field Descriptions

| Field | Description |
| --- | --- |
| `EvalDate` | Date of the Assessment |
| `Code` | Product Code of the Index |
| `RelPeriod` | Describes the period relative to the Evaldate and the MaturityType, whereas 00 is the current period and 01 is the first full period; e.g.: Evaldate="15/01/2013", MaturityType="M" -\> RelPeriod="00" equals "Jan 2013" and RelPeriod="01" equals "Feb 2013". |
| `PPStart` | Start of the period |
| `PPEnd` | End of the period |
| `Value` | Value of the index |
